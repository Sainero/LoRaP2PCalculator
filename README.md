# Run and deploy your AI Studio app

This contains everything you need to run your app locally.



## Run Locally

**Prerequisites:** Node.js 18+ (Vite 6), Yarn


1. Install dependencies:
   `yarn install`
2. Run the app (localhost:3000):
   `yarn dev`
 Если ошибка то yarn add react-is@^19
